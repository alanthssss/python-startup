## 精简：五行代码的面试题-分小球
一个常见的面试题，体现Python的精简。
req:有10个球分别3红、3蓝、4白，现需要将这10个球放入这3个盒子，要求每个盒子至少有一个白球。
```python
import random # 导入random模块
lst = [['w'],['w'],['w'],]
for i in ['w'] + ['r'] * 3 + ['b'] * 3:
random.choice(lst).append(i)
print(lst) # 打印结果
```
